# Домашнее задание 1 - Программирование на Python

## Описание
Решение задач по управляющим конструкциям и коллекциям на Python для ВШЭ.

## Задачи

### Задание 1: Поиск средней буквы
Функция `find_middle_letters()` возвращает среднюю букву в слове:
- Одну букву, если количество букв нечётное
- Две буквы, если количество букв чётное

**Примеры:**
- 'test' → 'es'
- 'testing' → 't'

### Задание 2: MVP dating-сервиса
Функция `dating_service()` формирует пары из отсортированных списков:
- Сортирует списки имён по алфавиту
- Создаёт пары с одинаковыми индексами
- Выводит предупреждение, если списки разной длины
